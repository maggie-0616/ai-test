Fakta om GIT

Vad är GIT?

GIT är ett versionshanteringssystem som hjälper dig att hålla koll på ändringar i din kod. Du kan samarbeta med andra, gå tillbaka till tidigare versioner och undvika att förlora arbete. GIT används av utvecklare världen över och är en grundpelare i modern webbutveckling.

  • Spåra ändringar i kod
  • Samarbeta i team
  • Återställ tidigare versioner
  • Open source och gratis
